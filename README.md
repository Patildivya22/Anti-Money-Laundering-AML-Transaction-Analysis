# Anti-Money-Laundering-AML-Transaction-Analysis
"Interactive Excel Dashboard for Anti-Money Laundering (AML) analysis, detecting high-risk transactions, temporal fraud patterns, and regional risk exposure."
# Project Overview
This project is an end-to-end Data Analytics solution designed to identify and visualize suspicious financial activities. By analyzing 4,999+ transactions, I built a dynamic dashboard to help compliance teams detect potential money laundering patterns like Structuring, Layering, and Smurfing.
 # Business Insights
Peak Fraud Hours: Suspicious activities significantly peak at 11:00 AM, suggesting a specific time window for high-risk transfers.
Risk Exposure: Identified 30 High-Risk alerts totaling over $226,466.
High-Risk Channels: Cross-border transactions and ACH transfers are the primary modes for high-value risk activities.
Regional Hotspots: Higher transaction volumes were detected in international regions like UAE and Switzerland.

# Technical Implementation
I organized the backend data using professional Pivot Table (PT) structures to ensure seamless interactivity:
PT_Country_Analysis: Regional volume mapping.
PT_Risk_Distribution: Categorizing alerts into High, Medium, and Low risk.
PT_Hourly_Fraud_Trend: Time-series analysis of alerts.
PT_Payment_Method_Risk: Analyzing risk by payment modes.

# Key Features
Dynamic Slicers: Filter data by Country, Risk Level, and Payment Mode.
Automated KPIs: Real-time tracking of Total Amount, Transaction Count, and Risk Value.
Visual Storytelling: Uses specialized icons and color-coded charts for quick decision-making.
# How to Use
Download the AML_Dashboard_Project.xlsx file.
Open in MS Excel.
Use the Slicers on the left to explore different risk scenarios.
